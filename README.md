# aiml-project-vityarthi
#WEATHER APPLICATION CLI#

This project is a simple weather application made using python which runs in the terminal. The program takes a place name from the user and shows the current weather details of that place using an online API.

To run this project you need python installed on your system along with an internet connection. You also need the requests library which can be installed using pip install requests. After that you can open terminal in the project folder and run the program using python app.py.

The program is easy to use. When it runs it asks the user to enter the place name. After entering the name it displays different weather details like temperature weather condition humidity pressure wind speed and sunrise and sunset time. It also shows feels like temperature.

#Main features of this project

takes input from user in terminal
shows real time weather data
displays multiple weather details
handles wrong place name
allows user to exit the program
